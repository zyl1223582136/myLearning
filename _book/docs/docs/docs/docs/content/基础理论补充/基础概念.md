# 基础概念

## 指示函数

### 定义

 在[集合论](https://baike.baidu.com/item/集合论/494533)中，**指示函数**是定义在某[集合](https://baike.baidu.com/item/集合/2908117)*X*上的[函数](https://baike.baidu.com/item/函数/301912)，表示其中有哪些元素属于某一子集*A*。
